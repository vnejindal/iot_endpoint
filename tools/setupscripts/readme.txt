This directory contains the tools like setupscripts and other software packages as they are created and released. 
Description: 
1. releases - contains release software packages 
2. setupscripts - contains package template and helper scripts: 
   -- pkg_template - contents of this directory should be copied to a new package and its scripts should be modified accordingly
   -- swpackage.sh - this script is used for creating a software package. It releases two files -- Software package archive (in .tar.gz format) and .sha file containing SHA512 of release package.

3. pktcapture - software package for installing wireshark 
4. webaccess - tool to enable browser based webaccess for tty and remote desktop
